## **Setup**

Make sure you have python & virtualenv package
```
pip install virtualenv
```
Create python venv
```
python3 -m venv venv
```

Active venv
```
source venv/bin/activate
```
Activate venv (windows)
```
.\venv\Scripts\activate
```
Install requirements
```
$ pip install -r requirements.txt
```

Run:
```
$ python main.py
```
